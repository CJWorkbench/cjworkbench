Introduction to data types

Introduction

Data types are used to tell the computer how to interpret and use values.

Different columns have different types: text, numbers, dates, locations, and so on. Some operations only work on specific types. For example, you can’t subtract one text value from another.

In this lesson we will use data on reported police incidents in New York to learn how Workbench guesses the types of incoming columns, and how to convert columns between types.

Data types

You may have noticed that Workbench shows the type of each column below the column name. Workbench supports 3 types:

Text

Also known as “string”, this type can contain any character and be of any length. Mixed numbers and letters are interpreted as text.

Number

This type is used for both whole numbers (such as 8, 22 or 268) and numbers which values have digits after the decimal point (such as 1.5 or 3.14).

Date & times

A date and a time together. In can be written in many formats, such as “23 January 2019 11:42”. Right now date & time in Workbench is shown as “2019–01-23 11:42:00”
