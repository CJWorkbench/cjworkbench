Data

The dataset we’re using is a prepared version of the data published and documented by the City of New York here.
It contains all police incidents in January and February 2018 in NYC. For the purpose of this lesson narrowed it down to a few select columns.

As usual, go through your list of questions to make sure you understand the data, and decide if it needs cleaning.

Instructions

Using the data dictionary, answer the following questions for yourself:

  1. What does each row represent?
  2. What's in each column?
  3. Are there any columns, values or codes you don't understand?
  4. Are there missing or obviously wrong values?
  5. Does the data need to be cleaned?

Group by date

Our first question is ‘How many incident were reported each month, in the entire city?

Instructions

  1. In step 6, add Group and select the column Complaint date.
  2. Check Group dates and select by month as granularity.
  3. Name the output columnCount of incidents.
  4. Add one more operation, and this time calculate the average donation per state. Name the new column Average amount.

Date granularity

Instructions
