Extreme and average values

Clean a messy dataset and format it in order to build a chart series

Introduction

Many stories are based on minimum or maximum values. This is the idea behind questions such as: What is the most expensive item in the budget? Which city is the safest? Which school has the best test scores? The average is one way of calculating a "typical" value that is useful to summarize data. How much CO2 emissions does a country produce in a year? In this lesson we will work with international data on CO2 emissions from 2000 to 2016.

1. Dropping empty rows and columns

For the purpose of this lesson, the dataset is hosted on Workbench.

Instructions

  1. Add the data source 'Add from URL'.
  2. Paste in the following URL and click Update to load the data.
    http://production-static.workbenchdata.com/data/population_growth_data.csv

    The two columns at the far right of the dataset are empty.

  3. Add the step 'Drop empty columns.'
  4. Add the step 'Drop empty rows'.
  5. In Drop empty rows, select columns MetroArea and Population. The rows missing values in those columns will be deleted.
