Introduction

As the links between climate change and health become more well-documented, pollution is a useful indicator of how different populations are impacted. This tutorial uses air quality data from the Environmental Protection Agency (EPA) to identify which major U.S. cities have outdoor particle 2.5 (or PM2.5, referring to inhalable particles smaller than 2.5 micrometers) pollution levels and/or outdoor particle 10 (or PM10, referring to inhalable particles smaller than 10 micrometers) pollution levels that are above the federal standards.

This tutorial was created by Betsy Ladyzhets in association with a feature for The Open Notebook on data journalism for science writers.

Obtaining the data

We will load the data for this story from the EPA’s Air Quality Statistics Report data feature, using 2018 data because 2019 data has not yet been finalized. We will exclude “exceptional events” so that outliers do not skew our analysis.

Learn more about the EPA air quality data report here.

Instructions

  1. Navigate to the EPA's Air Quality Statistics Report Data feature.

  2. Select 2018 and United States. Group results by city, and exclude exceptional events.
    Download the resulting CSV file for this data query.
  3. Add the step Upload as your data source for this workflow, and upload the CSV file you just downloaded.

The first step in cleaning these data is removing columns with extraneous information.
In this case, we've chosen to focus on particle pollution; this type of pollution is linked to polluters such as construction sites and fires, and poses distinct health risks. Thus, all other columns may be removed.

Cleaning the data 2/2

The next cleaning step for this dataset is to split the “CBSA” (core-based statistical area) column into city names and state names, to make the data easier to present later.
Finally, we will convert all the figures in the PM value columns so that Workbench recognizes these figures as numbers, and we can use formulas on them.

Using formulas

In order to identify which cities have PM pollution above EPA standards, we will use formulas to tag rows in the dataset.

Formulas are simple programming functions which may perform calculations on data points, organize them, or combine one dataset with another. Microsoft Excel is the classic program for formulas, and many journalists have their first data wrangling experience in either this program or its web-based sibling, Google Sheets.

This list by Adept Marketing’s Katie Cunningham, of ten key Excel functions to know, can give you an idea of the basics.

Workbench has the capacity to analyze data with formulas in the languages of Excel and Python. In this project, we will use an Excel logic formula called IF. This function automates logical comparisons between values: IF a value in another cell meets a certain condition, THEN the formula will return a predetermined output value. If the condition is not met, the formula will return a different predetermined value, which may be set to be a blank cell (or simply the word “FALSE”).

Filtering the data 1/2

In this project, we will use the IF formula to identify which cities have PM2.5 pollution above the EPA standard for 98th percentile 24-hour pollution (35 μg/m3), and which cities have PM10 pollution over the EPA standard for 2nd max 24-hour pollution (150 μg/m3). The full table of EPA standards is available here.

First, we will streamline the dataset further by using a filter to remove all rows with blanks in these two columns.

Next, we will tag the cities with high PM2.5 pollution. Our IF formula will determine the following: IF a city’s PM2.5 98th percentile 24-hour pollution value is higher than 35 μg/m3, THEN the output column will report “above.” For all other cities, the result column will report a blank.

Instructions

  1. Add the step Filter by condition.
  2. Set the filter so that if the cell in column PM10 Mean 24-hr is null OR the cell in column PM2.5 98th Percentile 24-hr is null, then delete .
  3. Add the step Formula, and select Excel.
  4. Add the following formula:
    =IF(D1>35, "above", "-" )
    Select Apply to all rows and name the output column PM 2.5 - EPA standard.

Filtering the data 2/2

We will repeat the formula step to tag cities with high PM10 pollution. Our IF formula will determine: IF a city’s PM10 2nd max 24-hour pollution value is higher than 150 μg/m3, THEN the output column will report above.

Now that we’ve set up the “above” tags, we can filter the dataset to only show us cities which have pollution above the EPA standard, either for PM2.5 or PM10.

Our final sorting step will rank the cities by their pollution values.

Instructions

  1. Add the step Formula, select Excel and apply the formula =IF(F1>150, "above", "-") to all rows. Name the output column PM 10 - EPA standard.
  2. Add the step Filter by condition, and filter as follow: IF the column PM 2.5 - EPA standard contains above OR the column PM 10 - EPA standard contains above, then Keep.
  3. Add the step Sort. Sort by PM2.5 98th Percentile 24-hr, then by PM10 2nd Max 24-hr, both by descending order.

Presenting the data 1/2

What is the best way to present these data? We’re not showing a percentage comparison, so a pie chart doesn't work, and we're not showing a trend, so a scatter plot doesn't work. A bar chart clearly allows readers to compare both across cities and between the PM2.5 and PM10 values.

In a more advanced version of this figure, readers might be able to click on cities to see more information about them, including other pollution values and context for the data. Context could include: what industries or events have contributed to pollution in this city? What is the local government doing to combat this issue?

Presenting the data 2/2
