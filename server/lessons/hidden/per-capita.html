Per capita and crime rates

Introduction

Which city has more coffee shops: Seattle or New York? New York is a lot larger, so it almost certainly has more coffee establishments, but Seattle’s coffee culture means that there may be more establishments per person.

Many numbers are best interpreted by comparing them to the overall population, giving a per-person rate. This is known as a per capita number (“capita” is Latin for “head”.)

In this lesson you’ll learn how to calculate per capita numbers using a simple spreadsheet formula on FBI crime data.

Crime per capita

Let’s reframe our question: ‘Which state has the most violent crimes per inhabitant?’

The formula to calculate ‘How many X per Y’ is X divided by Y, or:

=(X/Y)

Instructions

  1. Add Formula.

    Formulas must be written by referencing cells using column letters and rows.

  2. Type in:
    =(C1/B1)
  3. Name the output of the formula Violent crimes per capita
  4. Sort this new column in decreasing order.

    Alaska now comes on top. It has seen 6,133 violent crimes for a total population of 739,795 - or 0.008 violent crimes per inhabitant. That is twice as much as California, which now ranks #15.

Relative crime rates

These per capita crime rates are quite small: Alaska has only 0.008 violent crimes per person.
Because of this, crime rates are often reported as ‘crimes per 100,000 people’.

Instructions

  1. Edit your formula in step 3 to
    =(D1/B1)*100000

    If you look at the FBI source, you’ll see this number reported in the table.

  2. Rename the output of your Formula Crimes per 100,000.
