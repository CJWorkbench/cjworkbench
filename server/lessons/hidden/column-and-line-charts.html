Column and line charts

Introduction

Not all data stories need a data visualization. If you have only one or two numbers, they can probably just appear as text in the final story.
But with more data, charts are essential to quickly convey large amounts of information. In this lesson we will work with a prepared version of the same dataset, containing the 10 largest C02 emitters since 2010.

Column chart

Let’s first compare how much C02 has been produced by each country in 2016, using a column chart.

Instructions

  1. Add the step Column Chart in step 2.
  2. Select the column Country as the horizontal axis (X axis).
  3. Select the column Average 2000/16 as the vertical axis (Y axis)
  4. Are there missing or obviously wrong values?
  5. Does the data need to be cleaned?

Averages

Let's start by asking our data: What is the average emission per country, across all available years?
The average of a number (n) of values (v) is the sum of all values divided by the number of values:

=(Value1+Value2+Vn) / n

Instructions

  1. In step 2, add Calculate
  2. Set the operation to Average
  3. Select all available years columns
  4. Name the output column Average emission per year.

    The new column is displayed at the far right of the table.

Extreme values

Let's now ask: What is the most each country has emitted in any given year?

Instructions

  1. In step 3, add another Calculate.
  2. Set the operation to ‘Maximum’.
  3. Select available years columns and name the output Maximum 2000/16.

    The new column is displayed at the far right of the table.

Sorting

Another way to find extreme values is simply to sort the table. Let's ask our data: Which country has emitted the most C02 on average, across all available years?

Instructions

First, let's make our table easier to navigate

  1. In step 4, add 'Select columns', and keep columns Country,Continent and Average 2000/16.
  2. Using the column menu, sort Average 2000/16 in descending order.

Sorting by multiple columns

Which country is the largest CO2 emitter per continent? To find out, we need to sort first by Continent, then by Country.

Instructions

  1. Edit the existing step 4 to sort by Continent
  2. Still in step 4, click +ADD to add a new sort. This time, sort by Country in descending order.Keep top ‘1 row’, under the second sort.
  3. Keep the top ‘1 row’.
