Extreme and average values

Clean a messy dataset and format it in order to build a chart series

Introduction

Many stories are based on minimum or maximum values. This is the idea behind questions such as: What is the most expensive item in the budget? Which city is the safest? Which school has the best test scores? The average is one way of calculating a "typical" value that is useful to summarize data. How much CO2 emissions does a country produce in a year? In this lesson we will work with international data on CO2 emissions from 2000 to 2016.

Data

In this lesson we will use data by the U.S. Energy Information Administration, published and documented here.

Instructions

Take a look at this dataset and answer these questions for yourself

  4. Are there missing or obviously wrong values?

Averages

Let's start by asking our data: What is the average emission per country, across all available years?
The average of a number (n) of values (v) is the sum of all values divided by the number of values:

=(Value1+Value2+Vn) / n

Instructions

  1. In step 2, add Calculate
  2. Set the operation to Average
  3. Select all available years columns
  4. Name the output column Average emission per year.

    The new column is displayed at the far right of the table.

Sorting

Another way to find extreme values is simply to sort the table. Let's ask our data: Which country has emitted the most C02 on average, across all available years?

Instructions

  2. Using the column menu, sort Average 2000/16 in descending order.

Sorting by multiple columns
