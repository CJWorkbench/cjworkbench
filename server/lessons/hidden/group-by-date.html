Intro to data types

You may have noted that each column in any data table contains a ‘type’ of data. Data types are used to tell computer how to interpret and use values. Workbench supports 3 types:

Text
(also known as alphanumeric strings).

Number
Integers are whole numbers, such as 8,22 or 268. Floating point values are fractional values (such as 1/3). They are represented in a decimal number format: 1/3=3.333

Dates & times
Is used to to interpreted as a reference in time, which can vary depending on location. It can be displayed in many formats, such as YYYY-MM-DD (2019–01-23), 23 January 2019.
