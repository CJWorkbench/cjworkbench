In this first lesson we will go over the first questions you should ask about every data before starting your analysis. Then we will use international C02 emmissions data to do a complete story, including analysis and visualization.

Interviewing the Data

We pre-loaded data on international C02 emmissions, published by the U.S. Energy Information Administration here.

Instructions

Take a look at this dataset and answer these questions for yourself

  1. What does each row represent?
  2. What's in each column?
  3. Are there any columns, values or codes you don't understand?
  4. Are there missing or obviously wrong values?
  5. Does the data need to be cleaned?

Averages

One of the first things you can do with data is describe or summarize it. The average is one way of computing a "typical" value.

Let's ask "What volume of C02 is emitted by each country in a year?". CO2 emissions fluctuate, so instead of only using the last year of data, we will calculate the average of the last five years.

The average is the sum of all the values divided by the number of values:

=(Value 1+Value 2+...+Value N) / N

In this lesson we will use the step Calculate to apply this formula automatically.

Instructions

  1. Add the step Calculate
  2. Set the operation to Average
  3. Select columns 2012 to 2016
  4. Name the output column Average emission per year.
  5. Click the orange play button to apply your settings.

    The new column is displayed at the far right of the table.

First chart

Many data-driven stories involve a visualization. You can add a visualization step at any point to view the content of a table as a chart.

Let’s visualize the averages we’ve just calculated.

Instructions

  1. Click Add Step and type chart into the search box.
  2. Choose Column Chart
  3. Select the Country column for the X axis.
  4. Select the Average emission per year column for the Y axis
