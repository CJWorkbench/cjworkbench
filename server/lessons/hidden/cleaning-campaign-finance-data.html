Cleaning campaign finance data

Introduction

Campaign finance is one of the classic topics for data journalism. Unfortunately, campaign finance data often requires a lot of cleaning before you can answer basic questions, such as “Who donated the most money?”

In this lesson we will use data on contributions to a political campaign in the state of New York. We will extract the state from donor addresses, so that we can group donations by state. This requires splitting a column of text and removing unwanted information, which are basic cleaning skills you will use many times.

Data

Let's analyze campaign contributions to Andrew Cuomo’s 2014 re-election campaign in the State of New, published by the Board of Election of the State of New York, here.

Our goal is to find which state donated the most to his campaign. Out of state campaign contributions could reveal previously hidden business or political connections.

Splitting columns

Our first task is to extract the two letter state abbreviations in the Contributor column.
We need to store the state by itself in a new column, so that we can apply Group to it.

Instructions

  1. Select and keep the columns Contributor and Amt.
  2. Add Split column and select the column Contributor.
  3. Choose the method Extract x characters from right.
  4. Set X to 8, which is the total number of characters used by the post code (5) the state abbreviation (2) and the space in between them.
  5. Name the new column State.

Cleaning text

We now have the zip code plus the state in State. There are different ways to remove the extra information, but one of the simplest is to keep only letters.

Instructions

  1. Add Clean text, and apply it to the State column.
  2. In the 'Characters' section of the step, keep only letters (Deleting numbers would leave the space).

    Now we have a clean “State” column. Our table is ready to be analyzed!

  3. With step 4 selected, rename amt to Amount by editing the column name in the table. This will create a step amd log your edit.

Grouping

Next let's calculate the amount that has been donated in each state, using Group.

Instructions

  1. Add the step Group and apply it to State.
  2. Set the operation to SUM, and choose the Amt column as the target.
  3. Name the new column Total amount.

    We can now see the total amount of contributions from donors in each state.

Sort and chart

Let’s visualize these results.

Instructions

  1. Select Sort descending from the Total amount column menu .
  2. Add a Column Chart.
  3. Set the X Axis to State, and the Y axis to Total amount.
