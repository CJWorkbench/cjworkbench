Averages

One of the most basic things you can do with data is describe or summarize it. Country CO2 emissions fluctuate, so instead of looking at just the last year, let’s take the average of the last five years.

The average is one way of computing a "typical" value. It’s the sum of all the values divided by the number of values N:

=(Value 1+Value 2+...+Value N) / N
In this lesson we will use a ‘Calculate’ step to apply this formula automatically.

Instructions

  1. In step 2, add Calculate
  2. Set the operation to Average
  3. Select columns 2012 to 2016
  4. Name the output column Average emission per year.

    The new column is displayed at the far right of the table.
