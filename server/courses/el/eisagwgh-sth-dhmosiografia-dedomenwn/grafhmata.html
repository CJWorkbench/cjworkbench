Γραφήματα στηλών και γραφήματα γραμμών

Μάθετε πώς να χρησιμοποιείτε διαφορετικούς τύπους γραφημάτων και να κάνετε αντιμετάθεση για να ετοιμάσετε τα δεδομένα για γράφημα.

Εισαγωγή

Δεν είναι απαραίτητη η οπτικοποίηση των δεδομένων για όλες τις ιστορίες με δεδομένα. Αν έχετε μόνο έναν ή δύο αριθμούς, ίσως αρκεί να εμφανίζονται στην τελική ιστορία με τη μορφή κειμένου. Αν υπάρχουν περισσότερα δεδομένα, τα γραφήματα είναι απαραίτητα για να δώσετε στο κοινό σας να αντιληφθεί γρήγορα μεγάλες ποσότητες πληροφοριών.

Τα γραφήματα είναι επίσης χρήσιμα ως ενδιάμεσο βήμα στην ανάλυσή σας. Μπορείτε να χρησιμοποιήσετε γραφήματα για να εντοπίσετε σημεία ενδιαφέροντος στα δεδομένα σας χωρίς να είναι απαραίτητο να τα δείξετε στους αναγνώστες σας.

Σε αυτό το μάθημα θα εργαστούμε με το ίδιο σύνολο δεδομένων που χρησιμοποιήσαμε στο πρώτο μάθημα, το οποίο περιέχει δεδομένα για τις εκπομπές CO2 ανά χώρα από το 2010.

Πότε θα κάνουμε αντιμετάθεση

Ας συγκρίνουμε με οπτικοποίηση την ποσότητα CO2 που εκπέμπει η Κίνα με εκείνη που εκπέμπουν άλλες χώρες.

Αλλά, πρώτα απ' όλα, ας κοιτάξουμε καλύτερα τον πίνακα.
Στο Workbench, τα γραφήματα σχεδιάζονται με την επιλογή δεδομένων σε στήλες.

Δεν μπορούμε να διαλέξουμε την China γιατί τα δεδομένα για την Κίνα εμφανίζονται σε μία σειρά που διασχίζει πολλές στήλες. Ομοίως, δεν υπάρχει μία στήλη Year για να την επιλέξετε.

Αν θέλουμε να σχεδιάσουμε μεταβλητές που εμφανίζονται σε σειρές, πρέπει να τις μετατρέψουμε σε στήλες. Αυτό ονομάζεται αντιμετάθεση.

Πώς λειτουργεί η Αντιμετάθεση

Η Αντιμετάθεση μετατρέπει τις σειρές σε στήλες και τις στήλες σε σειρές. Έχει σαν αποτέλεσμα η κεφαλίδα να αλλάζει θέση με την αριστερή στήλη του πίνακα.

Δεδομένου ότι η αριστερή στήλη του σχήματος 1 γίνεται κεφαλίδα στο σχήμα 2, συχνά θα πρέπει να μετονομάσετε τη νέα αριστερή στήλη.

Η Αντιμετάθεση εν δράσει

Ας κάνουμε αντιμετάθεση στον πίνακα και ας σχεδιάσουμε σε γράφημα την ποσότητα του CO2 που εκπέμπει η Κίνα στο πέρασμα των χρόνων.

Πρώτα, πρέπει να διαγράψουμε τη στήλη Continent. Αν κάνουμε αντιμετάθεση στη στήλη (δοκιμάστε το!), θα εμφανιστεί σαν πρόσθετη μη αριθμητική σειρά και θα εμφανίσει σφάλμα όταν θα προσπαθήσουμε να σχεδιάσουμε γράφημα.

Οδηγίες

  1. Κάντε κλικ στο μενού της στήληςContinent και επιλέξτεΔιαγραφή στήλης
  2. Προσθέστε ένα βήμα Αντιμετάθεση και ονομάστε τη νέα στήλη αριστεράYear (Έτος)

    Τώρα μπορείτε να δημιουργήσετε γράφημα των εκπομπών αερίων της Κίνας.

  3. Προσθέστε ένα Γράφημα Στηλών, επιλέξτεYear για τον άξονα Χ καιChinaγια τον άξονα Y.

Πότε χρησιμοποιούμε γραμμικό γράφημα

Αν τα γραφήματα στηλών είναι μια καλή επιλογή για μια οπτική σύγκριση των ποσοτήτων CO2 που εκπέμπουν διαφορετικές χώρες σε δεδομένο χρονικό πλαίσιο, ένα γραμμικό γράφημα είναι καλύτερος τρόπος για να δείξουμε τις τάσεις στο πέρασμα του χρόνου. Οι γραμμές που συνδέουν τα σημεία δεδομένων βοηθάνε τους θεατές να αντιληφθούν καλύτερα τη διεύθυνση της μεταβολής.

Τα γραμμικά γραφήματα, ωστόσο, πρέπει να χρησιμοποιούνται πάντα με οριζόντιο άξονα που παριστάνει χρόνο. Το να τραβήξουμε μια γραμμή που να ενώνει την Κίνα με τη Ρωσία δεν θα είχε κανένα νόημα. Δεν υπάρχει κάποια ενδιάμεση τιμή και το γράφημα θα έδειχνε κάτι που δεν υπάρχει.

Γραμμικό γράφημα και χρονική σειρά

Ας οπτικοποιήσουμε τα ίδια δεδομένα, αλλά σε γραμμικό γράφημα.

Οδηγίες

  1. Μετά το γράφημα στηλών, προσθέστε ένα βήμα Γραμμικό Γράφημα.
  2. Επιλέξτε τη μεταβλητή Yearγια τον άξονα Χ καιChinaγια τον άξονα Y.

    Ας συγκρίνουμε τώρα την ποσότητα CO2 που εκπέμπει η Κίνα με την ποσότητα που εκπέμπουν άλλες χώρες.

  3. Κατά το βήμα του Γραμμικού Γραφήματος, επιλέξτε και άλλες χώρες που θέλετε στον άξονα Y κάνοντας κλικ στο κουμπί προσθήκης

Εδώ τελειώνει το μάθημα!
