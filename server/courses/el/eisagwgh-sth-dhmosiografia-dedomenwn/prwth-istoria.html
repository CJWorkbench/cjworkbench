Πρώτη ιστορία

Μάθετε για τις πρώτες ερωτήσεις που θα πρέπει να απευθύνετε στα δεδομένα σας πριν ξεκινήσετε την ανάλυσή σας και δημιουργήσετε το πρώτο σας γράφημα.

Καλωσορίσατε!

Σε αυτό το πρώτο μάθημα θα εξετάσουμε τις πρώτες ερωτήσεις που θα πρέπει να κάνετε σχετικά με όλα τα δεδομένα προτού ξεκινήσετε την ανάλυσή σας. Στη συνέχεια θα χρησιμοποιήσουμε τα δεδομένα για τις διεθνείς εκπομπές CO2 για να δημιουργήσουμε μια πλήρη ιστορία, η οποία θα περιλαμβάνει ανάλυση και οπτικοποίηση.

Ανάλυση των δεδομένων

Στο 1ο βήμα, κάναμε προφόρτωση δεδομένων για τις διεθνείς εκπομπές CO2 τα οποία δημοσιεύτηκαν από την Αμερικάνικη Υπηρεσία Πληροφόρησης για την Ενέργεια (U.S. Energy Information Administration) εδώ. Η μονάδα μέτρησης είναι η MMT (Million Metric Tons = εκατομμύρια μετρικοί τόνοι).

Οδηγίες

Κοιτάξτε αυτόν τον πίνακα και απαντήστε τις ακόλουθες ερωτήσεις για δικό σας λογαριασμό:

  • Τι αντιπροσωπεύει κάθε σειρά;
  • Τι υπάρχει σε κάθε στήλη;
  • Υπάρχουν στήλες, τιμές ή κωδικοί που δεν καταλαβαίνετε;
  • Υπάρχουν κενές ή εμφανώς λανθασμένες τιμές;
  • Χρειάζεται να καθαριστούν τα δεδομένα;

Καθάρισμα

Υπάρχει μια κενή τιμή στη στήλη Continent (Ήπειρος) . Μπορείτε να προσθέσετε ή να επεξεργαστείτε δεδομένα με διπλό κλικ σε ένα κελί.

Οδηγίες

  1. Στον πίνακα κάντε διπλό κλικ στο κενό κελί δίπλα στην India και πληκτρολογήστε τον κωδικό για την Ασία, που είναι AS σε αυτό το σύνολο δεδομένων.

    Σημειώστε ότι το νέο βήμα προστέθηκε στη ροή εργασίας σας στη θέση 2, με καταγραφή των τροποποιήσεων που κάνατε.

    Το Workbench καταγράφει κάθε τροποποίηση που γίνεται για να μπορείτε να επαναφέρετε μια προηγούμενη έκδοση. Για να δείτε τα δεδομένα πριν από την πρόσφατη τροποποίηση, κάντε κλικ στο βήμα 1.

Μέσες τιμές

Ένα από τα πιο βασικά πράγματα που μπορείτε να κάνετε με τα δεδομένα είναι να τα περιγράψετε ή να τα συνοψίσετε. Ο μέσος όρος είναι ένας τρόπος υπολογισμού μιας "τυπικής" τιμής.

Έστω ότι ρωτάμε "Ποιος είναι ο όγκος CO2 που εκπέμπει κάθε χώρα στη διάρκεια ενός έτους;" Οι εκπομπές CO2 κυμαίνονται και έτσι, αντί να χρησιμοποιήσουμε μόνο τα δεδομένα του τελευταίου έτους, θα υπολογίσουμε τον μέσο όρο για τα τελευταία τέσσερα χρόνια που διαθέτουμε.

Ο μέσος όρος είναι το άθροισμα όλων των τιμών όταν διαιρείται με το πλήθος των τιμών:

=(Τιμή 1+Τιμή 2+...+Τιμή N) / N

Σε αυτό το μάθημα θα χρησιμοποιήσουμε ένα βήμα υπολογισμού για να εφαρμόσουμε αυτόν τον τύπο χωρίς να χρειάζεται να τον πληκτρολογήσουμε.

Οδηγίες

  1. Κάνε κλικ στην Προσθήκη βήματος, πληκτρολογήστε Υπολογισμός στο πλαίσιο αναζήτησης και επιλέξτε το βήμα
  2. Ορίστε τη λειτουργία στην επιλογή Μέσος όρος:

    Κάντε κλικ στο κουμπί για να εφαρμόσετε τις αλλαγές όταν χρειάζεται.

  3. Κατά το βήμα Υπολογισμός επιλέξετε τις στήλες 2013, 2014,2015 και 2016.
  4. Ονομάστε τη στήλη εξόδου Μέσος όρος 2013-2016

    Η νέα στήλη θα εμφανιστεί στο δεξιό άκρο του πίνακα.

Πρώτο γράφημα

Πολλές ιστορίες βασισμένες σε δεδομένα χρησιμοποιούν γραφήματα. Μπορείτε να προσθέσετε ένα βήμα Γράφημα για να δείτε τα περιεχόμενα ενός πίνακα με τη μορφή γραφήματος.

Ας οπτικοποιήσουμε τους μέσους όρους που μόλις υπολογίσαμε.

Οδηγίες

  1. Κάνε κλικ στο Προσθήκη βήματος και επιλέξτε Γράφημα στηλών
  2. Επιλέξτε τη στήλη Country για τον άξονα X.
  3. Επιλέξτε τη στήλη Μέσος όρος 2013-2016 για τον άξονα Y.
